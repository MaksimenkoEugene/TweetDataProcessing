The application was created using Microsoft Visual Studio 2017 (Community).

Was used:
- WPF application template;
- Autofac for Dependency injection(DI).

Main project named 'TweetDataProcessing.App' was created for input parameters and displaying results.
Result returns sorted(by date) and distinct(by id) datas .
For data processing was used recursion.
Implemented logger functionality for data processing.
